# **App Name**: Eterno em Mim

## Core Features:

- Diário Poético: Transformar as reflexões e memórias em entradas de diário detalhadas e poéticas.
- Exploração Emocional: Aprofundar os sentimentos e emoções do personagem principal, utilizando ferramentas de IA para detectar nuances e sugestões de escrita.
- Criação de Capítulos: Dividir a história em capítulos bem estruturados com títulos criativos.
- Visualização de Memórias: Exibir as memórias em um formato visualmente atraente.
- Análise de Sentimentos: Utilizar um tool de IA para analisar e exibir os sentimentos expressos no diário.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to evoke tranquility and remembrance.
- Background color: Very light gray (#F5F5F5) to provide a calm, unobtrusive backdrop.
- Accent color: Dusty Rose (#D8BFD8) for subtle highlights and interactive elements.
- Body font: 'Literata', a serif font providing a literary and slightly vintage feel.
- Use minimalist, delicate icons representing memories and emotions.
- Maintain a clean, airy layout with plenty of white space to enhance readability and reflect the peaceful tone.
- Incorporate subtle transitions and fades to create a gentle, reflective user experience.
